# 2. 対象画面

本書で定義する画面（危険予知システム）は以下の通りである。

| No   | 画面グループ | 画面ID  | 画面名称       | 概要                                                   |
| ---- | ------------ | ------- | -------------- | ------------------------------------------------------ |
| 1    | 000          | V000010 | ログイン       | ユーザーID・パスワードを用いてログインを行う。         |
| 2    | 000          | V000020 | OTP認証        | ログイン時の二要素認証（ワンタイムパスワード）を行う。 |
| 3    | 000          | V000030 | パスワード変更 | ログインユーザーのパスワードを変更する。               |
| 4    | 010          | V000040 | 入力画面       | 工事内容や作業情報を入力する。                         |
| 5    | 020          | V000050 | 出力画面       | RAGが予測したインシデント（危険予知結果）を表示する。  |
