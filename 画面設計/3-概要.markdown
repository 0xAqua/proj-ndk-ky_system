# 1. はじめに

## 1.1.  本書の目的

本書は、「危険予知システム」の画面設計について記述するものである。

本書は、システム開発におけるフロントエンドの実装、および機能に仕様を定義することを目的とする。

## 1.2. システム概要

本システムは、建設・工事現場における作業の安全性を向上させることを目的とした危険予知システムである。 

ユーザー（現場監督者や作業員）が工事内容（作業場所、使用機材、作業手順など）を入力画面から送信すると、RAG（Retrieval-Augmented Generation）が入力内容に基づき、関連する過去の事故事例やヒヤリハット情報を検索・分析する。

 その分析結果から、予測されるインシデント（危険）を出力画面に返す。 これにより、作業前に潜在的なリスクを特定し、安全対策を講じることを支援する。
